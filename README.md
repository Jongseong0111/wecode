## Git 명령어 정리

###  git init 

wecode 폴더에서 git을 사용할 수 있도록 초기

### git status

git의 상태를 확인

### git add .

git 상태 staging
 
### git commit -m "Create test.py"

git을 장바구니에 넣는 단계

### git branch -M Main

git branch 이름을 Master에서 Main으로 변경

### git remote origin repository

github과 나의 git을 연결

### git push origin main

git에서 commit한 내용을 연동된 repository로 push
